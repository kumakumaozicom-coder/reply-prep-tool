# 返信準備メーカー

ChatGPTやClaudeに貼るための「返信準備プロンプト」を作る、小さなローカル用Webページです。

相手から来た文章を貼り付けて、連絡手段、相手との関係、こちらが伝えたいことなどを入力すると、返信文案をAIに作ってもらうためのプロンプトを生成します。

## 使い方

1. `index.html` をブラウザで開きます。
2. 連絡手段を `メール`、`ショートメール / SMS`、`LINE` から選びます。
3. 相手から来た文章や、こちらが伝えたいことを入力します。
4. 右側に表示された返信準備プロンプトをコピーして、ChatGPTやClaudeに貼り付けます。

## 機能

- 入力内容に合わせた返信準備プロンプトの自動更新
- 連絡手段ごとのルール反映
- コピーボタン
- TXTダウンロード
- リセットボタン

## 注意事項

- AI APIは使いません。
- 外部サービスには送信しません。
- メール、SMS、LINEを自動で読み取りません。
- メール送信、SMS送信、LINE送信はしません。
- 入力内容はブラウザ内だけで扱います。

## ファイル構成

- `index.html`: 画面の構造
- `style.css`: 見た目の調整
- `script.js`: プロンプト生成とボタン操作
- `README.md`: この説明ファイル
